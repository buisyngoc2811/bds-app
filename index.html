Cài đặt Firebase
Nhập thông tin Firebase để đồng bộ dữ liệu giữa điện thoại bố và chị.
1

Tạo project Firebase (miễn phí)

Truy cập console.firebase.google.com → Đăng nhập Gmail → Nhấn "Add project" → Đặt tên "bds-gia-dinh" → Tạo project.

2

Bật Realtime Database

Trong project → menu trái chọn "Realtime Database" → "Create database" → Chọn vùng Singapore → Chọn "Start in test mode" → Done.

3

Lấy config của project

Vào Project Settings (biểu tượng ⚙️) → tab "General" → cuộn xuống phần "Your apps" → nhấn icon </> → Đăng ký app → Copy phần firebaseConfig.

4

Nhập config vào đây

Dán từng giá trị từ firebaseConfig vào các ô bên dưới:

⚠️ Vui lòng nhập đầy đủ apiKey, projectId và databaseURL
